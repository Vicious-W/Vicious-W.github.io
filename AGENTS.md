# AGENTS.md

个人主页项目的 agent 上手说明。
本文件供所有 AI 编码 agent 读取：Codex 原生读取；Claude Code 通过 `CLAUDE.md` 里的 `@AGENTS.md` 导入读取。
**本文件刻意保持精简**，只作索引；完整内容在 `PROJECT.md`。

## 第一件事
**先读仓库根目录的 `PROJECT.md`** —— 它包含完整的项目目标、技术决策、当前进度和历史。

## 项目速览
- 个人主页，**静态网站**（纯前端，无后端、无数据库）。所有者的第一个开发项目。
- 目标：**强表现力 / 视觉冲击力 / 交互有趣**的个人主页。

## 关键工作约定
- 所有者**不做、也不打算学网页开发**：实现全权交给你。你负责技术选型与实现层面的所有决策，所有者只设定目标、判断观感与效果。
- 在静态站前提下把效果做到惊艳。
- 本项目由 git 管理，可以放心大胆地改动。
- **每次有实质进展或决策后，更新 `PROJECT.md`**（第 5 节「当前状态」＋第 7 节日志）。
- 这是网页项目，本环境装有 Playwright MCP 浏览器工具（navigate / 截图 / click / 读取 console 等）。
- **每次改动页面外观或行为后，必须用 Playwright MCP 实际渲染并查看效果，而不是只读代码：**
  启动本地服务 → 用浏览器工具打开页面 → 截图核对 → 检查 console 有无报错。
- 要用 **Playwright MCP** 的浏览器工具，不要改用 Bash 去跑 Playwright 脚本。
- 本地预览：在站点目录运行 `python3 -m http.server 8000`，再让浏览器工具访问 http://localhost:8000 。
